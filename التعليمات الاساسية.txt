


أريد منك إنشاء ثيم احترافي عالمي لمنصة سلة Salla باستخدام Twilight Engine، ويكون جاهزًا للبيع في متجر الثيمات بعد التخصيص والمراجعة.

الهوية :

  --color-primary: #6a1bce;     
  --color-secondary: #1e88e5;   
  --color-dark: #4a3bd1;        
  --color-accent: #00b9ad;

يمكنك تكوين تدرجات منها احترافيه عالميه

المطلوب العام:
- بناء ثيم Premium بمستوى عالمي، سريع، أنيق، قابل للبيع، ومصمم لرفع معدل التحويل Conversion.
- الالتزام الكامل بمعمارية Salla Twilight الرسمية.
- استخدام Twig في ملفات العرض.
- تنظيم المشروع حسب البنية الرسمية للثيم:
  - Layouts
  - Pages
  - Components
- استخدام ملفات ترجمة locales بصيغة JSON.
- استخدام ملف twilight.json لإعدادات الثيم وخيارات التخصيص.
- كتابة كود نظيف، منظم، وقابل للتوسعة.
- التركيز على الأداء، النظافة البصرية، وتجربة المستخدم على الجوال أولاً Mobile-first.

المكدس والتقنيات:
- Twig Templates للواجهات
- CSS/SCSS للتصميم
- JavaScript للتفاعلات
- تنظيم الأصول Assets بشكل احترافي
- الالتزام ببنية سلة الرسمية داخل src
- تجهيز المشروع بحيث يعمل مع salla theme preview

هيكلة المشروع المطلوبة:
- أنشئ المشروع وفق هذا التنظيم أو ما يعادله رسميًا في Twilight:
  - twilight.json
  - package.json
  - webpack.config.js إن لزم
  - src/
    - views/
      - layouts/
        - master.twig
      - pages/
        - index.twig
        - product/
        - customer/
        - blog/
        - brand/
        - cart/
        - page.twig أو ما يعادل الصفحات العامة المطلوبة
      - components/
        - header/
        - footer/
        - home/
        - product/
        - common/
    - locales/
      - ar.json
      - en.json
    - assets/
      - styles/
      - scripts/
      - images/
      - fonts/

مهم جدًا:
- لا تغيّر أسماء أو مسارات الصفحات الأساسية المطلوبة من Salla إذا كانت معرفة مسبقًا.
- ابنِ الصفحات بطريقة قابلة لإعادة الاستخدام عبر components.
- اجعل الصفحات ترث من master.twig.
- اجعل الثيم RTL/LTR ويدعم العربية والإنجليزية من البداية.

الهدف التجاري:
أريد ثيمًا يبدو مناسبًا للبيع في Salla Theme Marketplace، لذلك يجب أن يكون:
- احترافي جدًا
- مميز بصريًا
- سريع التحميل
- واضح للتاجر وسهل التخصيص
- مناسب للمتاجر السعودية والخليجية والعالمية
- يراعي تجربة الجوال بشكل ممتاز
- يبرز العروض والمنتجات بشكل يزيد المبيعات

الهوية البصرية المطلوبة:
- تصميم فاخر وحديث ونظيف

- Typography أنيقة وواضحة

- أقسام Homepage مبهرة لكن بدون ثقل أو ازدحام
- Footer احترافي ومتكامل

المظهر والتخصيص:
أريد الثيم قابل للتخصيص من التاجر عبر twilight.json، لذلك أنشئ إعدادات واضحة ومنظمة تشمل:
- ألوان الهوية الرئيسية والثانوية
- ألوان الأزرار
- ألوان الخلفيات
- Typography / أحجام الخطوط إن أمكن
- تفعيل/إخفاء أقسام الصفحة الرئيسية
- إعدادات البانرات
- إعدادات السلايدر
- إعدادات عرض المنتجات
- خيارات شكل البطاقات
- خيارات الهيدر والفوتر
- روابط التواصل الاجتماعي
- خيارات النشرة البريدية
- أي إعدادات مفيدة للتاجر بدون تعقيد

الترجمة:
- أنشئ ar.json و en.json
- لا تضع نصوصًا صلبة Hardcoded إلا عند الضرورة القصوى
- اجعل جميع النصوص القابلة للترجمة داخل locale files
- اجعل الترجمة العربية افتراضيًا أنيقة ومناسبة للسوق السعودي
- دعم RTL بشكل صحيح بالكامل

الأداء:
- أعطِ أولوية كبيرة للسرعة
- تقليل JS غير الضروري
- تنظيم CSS بعناية
- تجنب أي كود زائد
- تحسين الصور والأصول
- كتابة كود مناسب للمراجعة التقنية
- تجنب المؤثرات الثقيلة غير الضرورية
- اجعل Core UI سريعًا ومستقرًا

معايير الكود:
- نظّم الملفات بشكل واضح
- استخدم naming conventions جيدة
- افصل المسؤوليات بين layout/page/component
- علّق على الأجزاء المهمة فقط
- لا تكتب كودًا عشوائيًا أو مكررًا
- اجعل المكونات قابلة لإعادة الاستخدام
- حافظ على قابلية التوسع والتعديل

طريقة الإخراج:
- ابدأ أولاً بعرض شجرة الملفات المقترحة كاملة
- ثم أنشئ الملفات واحدًا واحدًا
- لكل ملف أعطني الكود الكامل
- لا تختصر ولا تقل “أكمل لاحقًا”
- إذا احتجت اتخاذ قرار تصميمي فاتخذه بنفسك بشكل احترافي
- لا تطلب مني تأكيدًا في كل خطوة
- اعمل كأنك Senior Salla Theme Engineer + UI/UX Designer + Conversion Designer

مهم جدًا:
- أريد الناتج نهائيًا بجودة بيع فعلية، وليس مجرد قالب تجريبي
- لا أريد تصميمًا تقليديًا محليًا فقط، بل مظهر عالمي راقٍ
- مع الحفاظ على ملاءمته للسوق العربي والسعودي
- اجعل النتيجة قابلة لاحقًا لإضافة screenshots ونشرها في Salla marketplace
